## 1.1 reflect

一个变量由一个pair来描述，形式如下

{"type": ... , "value": ...}，通过反射能获取变量的这两个属性

```go
package main

import (
	"fmt"
	"reflect"
)

func reflectNum(arg interface{}) {
	fmt.Println("type : ", reflect.TypeOf(arg))
	fmt.Println("value : ", reflect.ValueOf(arg))
}

func main() {
	var num float64 = 1.2345

	reflectNum(num)
}

```
